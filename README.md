# ShoppingMall

이 프로젝트는 Spring Boot, Thymeleaf, JPA 및 MySQL을 사용하여 진행하였습니다!

## 배포 중인 프로젝트

## 개발환경
- 운영체제: Windows 10
- 통합 개발 환경: IntelliJ IDEA
- 데이터베이스: MySQL, MySQL Workbench
- 웹 브라우저: Chrome
- 언어: Java 17
- 프레임워크: Spring Boot
- 내장 톰캣 서버 사용
- 기술: Spring Data JPA, JavaScript, HTML5, CSS3

## 도구 및 API
- 버전 관리: Git / GitHub
- UI 디자인: Bootstrap
- 로그인 API: Google, Daum, Naver
- 보안: Spring Security
